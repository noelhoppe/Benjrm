# Benjrm's Frontend

## Used Dependencies:
1. React + TypeScript
2. React Router
3. Tanstack Query
4. Tailwind CSS
5. shadcn/ui

## NPM Scripts:

Starting the development server with HMR (Hot Module Replacement):
```shell
npm run dev
```

Linting the codebase to find and fix issues:
```shell
npm run lint
```

Building the application for production:
```shell
npm run build
```

Previewing the production build locally:
```shell
npm run preview
```
